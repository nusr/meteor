# meteor

在 **react** ,**react-native** 中使用同一个 **meteor** 

## 安装依赖

```bash
yarn
```

or

```bash
npm i
```

## 启动 meteor 

```bash
npm start
```

## 启动 react 

```bash
npm start
```

## 启动 react-native android

1. 启动安卓模拟器或者连接安卓手机

```bash
npm run start:android
```

## 启动 react-native ios

1. 启动 IOS 模拟器或者连接苹果手机

```bash
npm run start:ios
```

## 功能

1. [x] react
1. [x] react-native android
1. [x] react-native ios
