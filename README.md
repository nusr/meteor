# meteor

在 **react** ,**react-native** 中使用同一个 **meteor** 

## 安装依赖

`npm i` or `yarn`

## 启动 meteor 

```bash
npm start
```

## 启动 react 

```bash
npm start
```

## 启动 react-native android

1. 启动安卓模拟器或者连接安卓手机
2. `npm run start:android`


## 功能

1. [x] react
2. [x] react-native android
2. [ ] react-native ios
